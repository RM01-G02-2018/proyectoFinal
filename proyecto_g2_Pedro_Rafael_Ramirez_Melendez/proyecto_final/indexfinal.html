Llega el nuevo Visual Studio Code 1.7 con interesantes mejoras


Aunque una de las herramientas más sencillas de Microsoft para editar texto plano es el clásico bloc de notas, esta aplicación puede llegar a resultar tan simple que, para ciertas tareas, está muy limitada. Por ello, muchos desarrolladores, e incluso la propia Microsoft, han creado editores de texto más completos que el bloc de notas para todos aquellos usuarios con necesidades tanto de un editor simple pero completo como de un IDE de programación, como es el caso de Visual Studio Code.

Visual Studio Code es un editor de texto y entorno de programación desarrollado por Microsoft, gratuito y de código abierto, creado bajo la marca “Visual Studio”. Este editor de texto se caracteriza por ofrecer un entorno totalmente personalizable a base de plugins a la vez que es capaz tanto de sustituir al clásico bloc de notas ofreciendo a los usuarios más funciones (como, por ejemplo, el uso de pestañas) como un entorno de programación compatible con prácticamente cualquier lenguaje gracias a los ya mencionados plugins, tanto creados por Microsoft como por la comunidad.

Con el fin de seguir creciendo, Microsoft ha lanzado recientemente el nuevo Visual Studio Code 1.7 que, teniendo muy en cuenta las peticiones de la comunidad, llega con un gran número de cambios y mejoras que, probablemente, serán del agrado de la mayoría.

Novedades del nuevo Visual Studio Code 1.7

Una de las novedades que ha sido incluida en esta nueva versión es la posibilidad de organizar los editores en vertical y horizontal en lugar de solo en vertical, de manera, que los usuarios puedan adaptarlo tanto a sus gustos como a sus necesidades (especialmente para aquellos que trabajan o con dos monitores o con un monitor en vertical). Además, si trabajamos con varios editores, ahora podemos desactivar fácilmente las extensiones para uno de ellos o para todos.

Otra de las novedades que incluye es una chuleta en PDF con todos los atajos de teclado de la herramienta de manera que, en vez de tener que memorizarlos, podamos tenerlos siempre a mano. Además, esta nueva versión cuenta con mapeo de atajos de teclado de Sublime Text y Atom.

En tercer lugar, esta nueva versión es totalmente compatible con JavaScript IntelliSense, instalando además todos los módulos y todas las dependencias tan pronto como se instala el módulo JS. También se ha mejorado la gramática para los lenguajes TypeScript y JavaScript.

Otra de las novedades más es el nuevo autocompletado para el código CSS insertado en el HTML, así como un gran número de mejoras en la depuración de errores en todos los lenguajes, aunque especialmente en Node.js.

Por último, ahora es posible ver todas las dependencias de las extensiones fácilmente para evitar problemas a la hora de instalarlas y activarlas.

Podemos descargar esta herramienta de forma totalmente gratuita tanto para Windows como para macOS y Linux desde su página web oficial. Sin duda una herramienta que no debe faltar en los ordenadores de aquellos que lo utilicen tanto para editar ficheros en texto plano como para programar en cualquier lenguaje de programación.

Hadron: una herramienta para crear y diseñar aplicaciones web fácilmente


Tal y como ha venido sucediendo a lo largo de los últimos años, todo lo relacionado con el mundo de la programación y el desarrollo de proyectos ha ido mejorando de manera ostensible, ya que el número de creadores de contenidos y profesionales del sector ha aumentado mucho en estos últimos tiempos.

Es por ello que cada vez podemos echar mano de más servicios y plataformas centradas en este tipo de tareas y que al mismo tiempo intentan facilitarnos el desarrollo de aplicaciones, incluso para los trabajos en grupo, como es el caso del que os vamos a hablar en estas líneas.

En concreto vamos a hacer referencia a Hadron una aplicación web que tiene como claro objetivo facilitar el desarrollo de proyectos de programación aportando ciertas funciones en este sentido. Así, nos ofrece un entorno de desarrollo de código muy visual, cómodo y atractivo de usar, todo ello vía web. Al mismo tiempo uno de los propósitos de Hadron es el poder unificar en una única plataforma, el trabajo tanto de los programadores como tal, y el de los diseñadores, algo no demasiado frecuente en estos entornos.

De este modo lo que se pretende lograr con este trabajo conjunto, es que a medida que la nueva herramienta va siendo creada, al mismo tiempo los diseñadores le pueden ir dando forma, y así no tener que esperar al final del proyecto, como sucede normalmente en muchas otras ocasiones. Esto se traduce en que por medio de esta aplicación, tendremos la oportunidad de empezar a trabajar en nuestros proyectos de código en HTML y CSS simple, para más adelante, poco a poco, ir introduciendo elementos más relacionados con el diseño.

Ventajas que nos ofrece Hadron en el desarrollo de proyectos de programación



Esto es algo que lograremos gracias a la incorporación de diversos elementos web o Javascript, mientras que al mismo tiempo tendremos la oportunidad de utilizar CSS Grid, Color Picker o Shadows.

Y claro, de manera paralela ir desarrollando el proyecto HTML o insertar varios elementos ya desarrollados y que la propia plataforma pone a nuestra disposición. De todo esto podremos hacer uso dentro de un entorno muy amigable y funcional, lo que nos ayudará mucho a la hora de trabajar en el código de nuestras propias aplicaciones.

Para terminar también merece la pena saber que Hadron nos va a permitir el poder trabajar sin conexión a Internet para después sincronizar el nuevo código generado al proyecto en Hadron cuando ya tengamos conexión a la Red, todo ello mediante el uso de un dominio propio que nos proporciona la plataforma.

Tipos de lenguajes de programación


Existen dos tipos de lenguajes claramente diferenciados; los lenguajes de bajo nivel y los de alto nivel.

El ordenador sólo entiende un lenguaje conocido como código binario o código máquina, consistente en ceros y unos. Es decir, sólo utiliza 0 y 1 para codificar cualquier acción.

Los lenguajes más próximos a la arquitectura hardware se denominan lenguajes de bajo nivel y los que se encuentran más cercanos a los programadores y usuarios se denominan lenguajes de alto nivel.

Lenguajes de Bajo Nivel ver mas...

Son lenguajes totalmente dependientes de la máquina, es decir que el programa que se realiza con este tipo de lenguajesno se pueden migrar o utilizar en otras maquinas.

Al estar prácticamente diseñados a medida del hardware, aprovechan al máximo las características del mismo.

Dentro de este grupo se encuentran:

  • El lenguaje maquina: este lenguaje ordena a la máquina las operaciones fundamentales para su funcionamiento. Cnsiste en la combinación de 0's y 1's para formar las ordenes entendibles por el hardware de la maquina. Este lenguaje es mucho más rápido que los lenguajes de alto nivel. La desventaja es que son bastantes difíciles de manejar y usar, además de tener códigos fuente enormes donde encontrar un fallo es casi imposible.
  • El lenguaje ensamblador es un derivado del lenguaje maquina y esta formado por abreviaturas de letras y números llamadas mnemotécnicos. Con la aparición de este lenguaje se crearon los programas traductores para poder pasar los programas escritos en lenguaje ensamblador a lenguaje máquina. Como ventaja con respecto al código máquina es que los códigos fuentes eran más cortos y los programas creados ocupaban menos memoria. Las desventajas de este lenguaje siguen siendo prácticamente las mismas que las del lenguaje ensamblador, añadiendo la dificultad de tener que aprender un nuevo lenguaje difícil de probar y mantener.

Lenguajes de Alto Nivel ver mas...

Son aquellos que se encuentran más cercanos al lenguaje natural que al lenguaje máquina. Están dirigidos a solucionar problemas mediante el uso de EDD's.

Nota: EDD's son las abreviaturas de Estructuras Dinamicas de Datos, algo muy utilizado en todos los lenguajes de programación. Son estructuras que pueden cambiar de tamaño durante la ejecución del programa. Nos permiten crear estructuras de datos que se adapten a las necesidades reales de un programa.

Se tratan de lenguajes independientes de la arquitectura del ordenador. Por lo que, en principio, un programa escrito en un lenguaje de alto nivel, lo puedes migrar de una máquina a otra sin ningún tipo de problema.

Estos lenguajes permiten al programador olvidarse por completo del funcionamiento interno de la maquina/s para la que están diseñando el programa. Tan solo necesitan un traductor que entiendan el código fuente como las características de la maquina.

Suelen usar tipos de datos para la programación y hay lenguajes de propósito general (cualquier tipo de aplicación) y de propósito especifico (como FORTRAN para trabajos científicos).

Lenguajes de Medio Nivel ver mas...

Se trata de un termino no aceptado por todos, pero q seguramente habrás oído. Estos lenguajes se encuentran en un punto medio entre los dos anteriores. Dentro de estos lenguajes podría situarse C ya que puede acceder a los registros del sistema, trabajar con direcciones de memoria, todas ellas características de lenguajes de bajo nivel y a la vez realizar operaciones de alto nivel.

Generaciones

La evolución de los lenguajes de programación se puede dividir en 5 etapas o generaciones.

  • Primera generación: lenguaje maquina.
  • Segunda generación: se crearon los primeros lenguajes ensambladores.
  • Tercera generación: se crean los primeros lenguajes de alto nivel. Ej. C, Pascal, Cobol
  • Cuarta generación. Son los lenguajes capaces de generar código por si solos, son los llamados RAD, con lo cuales se pueden realizar aplicaciones sin ser un experto en el lenguaje. Aquí también se encuentran los lenguajes orientados a objetos, haciendo posible la reutilización d partes del código para otros programas. Ej. Visual, Natural Adabes
  • Quinta generación: aquí se encuentran los lenguajes orientados a la inteligencia artificial. Estos lenguajes todavía están poco desarrollados. Ej. LISP
